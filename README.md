# Procurement Data Analytics & Spend Analysis

Comprehensive procurement analytics system with SAP integration for spend analysis and supplier performance tracking.

## Features

- **Spend Analysis**: Category-wise spend breakdown
- **Supplier Performance**: OTD rate, quality scores
- **Cost Savings**: Identify savings opportunities
- **SAP Integration**: SQL queries for data extraction
- **Vendor Ranking**: Multi-criteria supplier evaluation

## Tech Stack

- Python 3.8+
- pandas, numpy
- SQL (SAP HANA compatible)
- matplotlib for visualization

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```

## Project Structure

```
src/
  spend_analyzer.py         - Spend analysis module
  supplier_performance.py   - Supplier KPI tracking
  generate_sample_data.py   - Sample procurement data
main.py                     - Main execution script
```

## Key Metrics

- Total spend analysis across categories
- Supplier OTD rate tracking
- Cost reduction opportunities identified
